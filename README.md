# Quisori（キソーリ）

> 基礎から学べる学生向け学習サービス

学生向けの学習支援サービスです。

現在は数学を中心に、問題演習を通して基礎学力の定着をサポートすることを目的に開発しています。

⚠ 現在開発中のプロジェクトです。

## Website

https://sor-609.github.io/Quisori/index.html

## Features

- 数学問題の演習
- 分野ごとの学習
- 問題と解説の表示
- スマートフォン・PC対応

## Tech Stack

- HTML
- CSS
- JavaScript

## Motivation

私、sor-609は学生として、日々の学習の中でさまざまな学習サイトやサービスを利用してきました。

それぞれに優れた点がある一方で、「ここがもっとこうだったら使いやすいのに」と感じる場面も少なくありませんでした。

そこで、自分自身の学習経験を活かし、学生目線で本当に使いやすい学習サービスを作りたいと思い、Quisoriの開発を始めました。

現在は数学を中心に開発を進めていますが、将来的にはより多くの学生の学習を支援できるサービスへ成長させたいと考えています。
